# Iris Gestión

Sistema de gestión de inventario y ventas desarrollado en Node.js con frontend web simple.

## Funcionalidades

- Gestión de productos (alta, edición, activación/desactivación)
- Control de stock automático
- Registro de ventas con múltiples productos
- Anulación de ventas con devolución de stock
- Resumen diario por fecha
- Persistencia en archivos JSON
- Frontend web para uso cotidiano

## Tecnologías

- Node.js
- Express
- JavaScript (Vanilla)
- HTML / CSS
- JSON como persistencia

## Instalación

git clone https://github.com/guiyeIbarra/iris_gestion.git
